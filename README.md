# House Price Prediction using Machine Learning

---

## Overview
This project predicts house prices using Machine Learning techniques. It demonstrates a complete end-to-end ML pipeline including data preprocessing, feature selection, model training, evaluation, and deployment using a web interface.

---

## Features
- Data preprocessing and cleaning  
- Feature selection  
- Model training using Linear Regression  
- Model evaluation (MSE & R² Score)  
- Data visualization  
- Interactive web app using Streamlit  

---

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Seaborn  
- Streamlit  

---

## Dataset
Dataset sourced from Kaggle:  
**House Prices - Advanced Regression Techniques**

---

## Model Used
- Linear Regression  

---

## Model Performance
- Evaluated using:
  - Mean Squared Error (MSE)  
  - R² Score

## How to Run the Project

### 🔹 1. Clone the repository
```
git clone https://github.com/your-username/house-price-ml.git
cd house-price-ml
```

### 🔹 2. Install dependencies
```
pip install -r requirements.txt
```

### 🔹 3. Run Streamlit app
```
cd app
streamlit run app.py
```
