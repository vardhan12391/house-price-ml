House Price Prediction using Machine Learning
Overview

This project predicts house prices using machine learning regression techniques. It demonstrates an end-to-end ML pipeline including data preprocessing, feature selection, model training, and evaluation.

Technologies Used
Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn
Streamlit (for deployment)

Dataset

Dataset sourced from Kaggle
House Prices - Advanced Regression Techniques

Model
Linear Regression
Results
Evaluated using Mean Squared Error (MSE)
Achieved strong R² score indicating good model performance
Web Application

A simple interactive web app is built using Streamlit where users can input house features and get predicted prices.

Features
Data preprocessing and cleaning
Feature scaling
Model training and evaluation
Data visualization
Interactive prediction UI

Key Learnings
End-to-end ML pipeline
Regression modeling
Feature engineering basics
Model evaluation techniques
